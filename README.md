## Test Automation Framework

An end-to-end automation framework for web and API testing, designed to improve testing efficiency and system reliability.

###🔹 Key Features
- Automated UI and API test cases
- Reusable test suites for critical workflows
- API validation for data consistency
- Data-driven testing for multiple scenarios
- CI/CD integration for automated execution

###🔹 Impact
- Reduced manual testing effort by ~60%
- Improved release quality and system reliability

###🔹 Tech Stack
C#, .NET, Selenium, Playwright, SQL Server, Postman, CI/CD
